# CURRENT\_TIMESTAMP

Returns the current year, month, day and time as a `TIMESTAMP` value, formatted as YYYY-MM-DD hh:mm:ss.

## Syntax
{: .no_toc}

```sql
CURRENT_TIMESTAMP
```

## Example
{: .no_toc}

```
SELECT
    CURRENT_TIMESTAMP;
```

**Returns**: `2022-10-12 19:39:22`
